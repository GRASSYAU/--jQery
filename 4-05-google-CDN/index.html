台北 jQuery 教學課程推薦： jQuery 非常簡單易學，即使用完全沒有寫過程式的網頁設計師，也能夠輕鬆學會 jQuery 前端網頁程式，製作出效果一流的商業等級前端網頁動態效果。本課程從網頁設計師的角度解說與教學，是 jQuery 的入門教學課程，為完全沒有程式基礎的設計師量身打造的 jQuery 網頁前端程式入門教學課程，從最基本的 jQuery 程式指令語法講起，先一步一步從最根本打好您的程式基礎，再討論「DOM 文件物件模 型」的觀念jQuery 與 CSS 的整合應用...讓您學會網頁前端程式的觀念和使用的精髓。從入門到精通，讓您的網頁設計能力大大的提昇！本課程的教學方式是以範例式為導向，上課時透過老師一步一步圖解超詳細的講解，讓您輕鬆快速學會 jQuery 各種指令的運用，jQuery 改良自原生的 JavaScript 程式，大大地簡化程式的寫法，原本使用 JavaScript 需要花上數十行才能完成的前端網頁效果，如今透過 jQuery 只需短短的幾行程式碼就能夠達到相同的前端網頁效果。上課所練習的範例都是目前網路上常見又實用的商業等級前端網頁互動效果，同學上課練熟之後就可以馬上就運用到您工作的網頁之中。
